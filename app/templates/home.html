How to start:
Manufacturing: create Products and Bill of Materials. Both support import from CSV.
Or skip Product creation and import a BOM with associated BOM Lines from CSV.
CSV must contain a top product (assembly) and products (component parts) for individual BOM Lines. See Help for samples.
If a product associated with a BOM Line is not found, it will be created automatically.
Products are uniquely identified by name, and duplicates are skipped.
Purchase: create Supplier(s).
Now that you have Suppliers and Products, you can create Purchase Orders either manually or import from CSV.
As soon as PO is marked as completed, the following records will be automatically created in the Inventory module:
  • Inventory Items for each product on a PO Line,
  • Stock Moves from Supplier to Warehouse.
Manufacturing: create an MO (Manufacturing Order).
Select a BOM for the MO. BOM Lines will populate automatically.
To mark an MO as Reserved, each BOM Line must have the required quantity of components available as a result of completed POs.
If the required quantity for each BOM Line is available, you may reserve the components.
This will automatically update Inventory Items and create Stock moves from Warehouse to Factory.
If an MO is cancelled prior to being completed, the these changes will automatically triggered in the Inventory module:
  • new Stock Moves from Factory back to Warehouse will be created,
  • Inventory Items will be updated
