How to start:

  1. In Manufacturing module: create Product(s) manually or bulk import from CSV.
  2. After you have Products you can create a BOM (Bills of Materials) and associated BOM Lines manually or import from CSV
    • If you prefer, skip step 1 and start by importing a BOM with its BOM Lines from CSV.
    • CSV must contain a top product (assembly) and products (component parts) for individual BOM Lines. See Help for samples.
    • If a product associated with a BOM Line is not found it will be created automatically.
    • Products are uniquily identified by name and duplicates are skipped
  3. In Purchase module: create Supplier(s).
  4. Now that you have Suppliers and Products, you can create POs (Purchase Orders) either manually or import from CSV.
    • PO has following states: Draft, Pending Approval, Approved, In Progress, Completed.
    • As soon as PO is marked as completed the following records will be automatically created in the Inventory module:
      • Inventory Items for each product on a PO Line,
      • Stock Moves from Supplier to Warehouse.
  5. In Manufacturing module: create an MO(Manufacturing Order).
    • Select a BOM for the MO. BOM Lines will populate automatically.
    • MO has following states: Draft, Pending, Reserved, In Progress, Completed.
    • To mark an MO as Reserved each BOM Line must have required quanity of components available as a result of POs completed above.
    • If required quanity for each BOM Line is available, you may reserve the components.
    • This will automatically update Inventory Items and create Stock moves from Warehose to Factory.
    • If MO is cancelled prior to being completed:
      • new Stock Moves from Factory back to Warehose will be created,
      • Inventory Items will be updated.
